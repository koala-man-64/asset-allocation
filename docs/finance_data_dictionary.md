# Finance Data Dictionary

| Column Name | Description |
| :--- | :--- |
| **symbol** | Unique ticker identifier for the company. |
| **total_revenue** | Total revenue generated from all operating activities. |
| **operating_revenue** | Revenue generated specifically from primary business operations. |
| **cost_of_revenue** | The direct costs associated with producing the goods or services sold (COGS). |
| **gross_profit** | Total Revenue minus Cost of Revenue. |
| **operating_expense** | Expenses incurred through normal business operations (OPEX). |
| **selling_general_and_administration** | specialized operating expenses: Selling, General, and Administrative (SG&A). |
| **general_and_administrative_expense** | Expenses related to the day-to-day operations of managing the business. |
| **salaries_and_wages** | Compensation paid to employees. |
| **other_gand_a** | Miscellaneous General and Administrative expenses. |
| **selling_and_marketing_expense** | Costs associated with marketing and selling products or services. |
| **research_and_development** | Expenses for research and development (R&D) activities. |
| **depreciation_amortization_depletion_income_statement** | Systematic allocation of the cost of tangible and intangible assets over their useful lives (Income Statement view). |
| **depreciation_and_amortization_in_income_statement** | Combined depreciation and amortization reported on the income statement. |
| **other_operating_expenses** | Operating expenses not categorized elsewhere. |
| **operating_income** | Profit realized from business operations (EBIT before non-operating items). |
| **net_non_operating_interest_income_expense** | Net value of interest income minus interest expense from non-operating sources. |
| **interest_income_non_operating** | Interest earned from investments or cash holdings. |
| **interest_expense_non_operating** | Interest paid on debt obligations. |
| **other_income_expense** | Combined other income and expenses. |
| **gain_on_sale_of_security** | Profit realized from the sale of securities. |
| **earnings_from_equity_interest** | Income earned from investments in other companies (equity method). |
| **special_income_charges** | Unusual or one-time income or charges. |
| **restructuring_and_mergern_acquisition** | Costs associated with restructuring or M&A activities. |
| **write_off** | Value of assets removed from the books (write-downs or write-offs). |
| **other_non_operating_income_expenses** | Miscellaneous non-operating income or expenses. |
| **pretax_income** | Earnings before income tax expenses. |
| **tax_provision** | Estimated amount of income tax needing to be paid for the current period. |
| **net_income_common_stockholders** | Net income available to common shareholders (after preferred dividends). |
| **net_income** | The total profit of the company associated with all shareholders. |
| **net_income_including_noncontrolling_interests** | Net income including the portion attributable to non-controlling interests. |
| **net_income_continuous_operations** | Net income derived from core, continuing business operations. |
| **net_income_extraordinary** | Income or loss from extraordinary, infrequent events. |
| **preferred_stock_dividends** | Dividends paid to preferred stockholders. |
| **diluted_ni_availto_com_stockholders** | Net income available to common stockholders adjusted for dilution. |
| **basic_eps** | Earnings Per Share calculated using the weighted average number of common shares. |
| **diluted_eps** | Earnings Per Share calculated using fully diluted shares outstanding. |
| **basic_average_shares** | The weighted average number of shares used for Basic EPS. |
| **diluted_average_shares** | The weighted average number of shares used for Diluted EPS. |
| **total_operating_income_as_reported** | Operating income as specifically reported by the company. |
| **total_expenses** | Sum of all expenses incurred. |
| **net_income_from_continuing_and_discontinued_operation** | Total net income including both continuing and discontinued operations. |
| **normalized_income** | Net income adjusted to remove the effects of seasonality or one-time events. |
| **interest_income** | Total interest income. |
| **interest_expense** | Total interest expense. |
| **net_interest_income** | Interest Income minus Interest Expense. |
| **ebit** | Earnings Before Interest and Taxes. |
| **ebitda** | Earnings Before Interest, Taxes, Depreciation, and Amortization. |
| **reconciled_cost_of_revenue** | Cost of Revenue adjusted or reconciled for analytical consistency. |
| **reconciled_depreciation** | Depreciation adjusted or reconciled for analytical consistency. |
| **net_income_from_continuing_operation_net_minority_interest** | Net income from continuing operations minus minority interest. |
| **total_unusual_items_excluding_goodwill** | Unusual items affecting income, excluding goodwill impairments. |
| **total_unusual_items** | Sum of all unusual or non-recurring items. |
| **normalized_ebitda** | EBITDA adjusted for non-recurring items. |
| **tax_rate_for_calcs** | Effective tax rate used for calculations. |
| **tax_effect_of_unusual_items** | The tax impact of unusual items. |
| **total_assets** | Sum of all current and non-current assets. |
| **current_assets** | Assets expected to be sold, consumed, or exhausted within one year. |
| **cash_cash_equivalents_and_short_term_investments** | Highly liquid assets including cash, equivalents, and short-term investments. |
| **cash_and_cash_equivalents** | Physical currency and highly liquid investments. |
| **cash_financial** | Cash held by financial institutions. |
| **cash_equivalents** | Short-term, highly liquid investments that are readily convertible to cash. |
| **other_short_term_investments** | Short-term investments not classified as cash equivalents. |
| **receivables** | Money owed to the company. |
| **accounts_receivable** | Money owed by customers for goods/services delivered on credit. |
| **gross_accounts_receivable** | Total accounts receivable before deducting allowances. |
| **allowance_for_doubtful_accounts_receivable** | Estimate of accounts receivable that may not be collected. |
| **inventory** | Goods available for sale and raw materials used to produce goods. |
| **raw_materials** | Basic materials used in the production of goods. |
| **work_in_process** | Goods in the process of production but not yet finished. |
| **finished_goods** | Completed goods ready for sale. |
| **current_deferred_assets** | Assets representing costs paid in advance (prepaids) or deferred tax assets (current). |
| **current_deferred_taxes_assets** | Tax credits or future tax benefits to be realized within a year. |
| **hedging_assets_current** | Current value of hedging instruments. |
| **other_current_assets** | Miscellaneous current assets not categorized elsewhere. |
| **total_non_current_assets** | Assets not expected to be converted to cash within one year. |
| **net_ppe** | Property, Plant, and Equipment net of accumulated depreciation. |
| **gross_ppe** | Total value of Property, Plant, and Equipment before depreciation. |
| **properties** | Value of land and properties owned. |
| **land_and_improvements** | Value of land and any improvements made to it. |
| **buildings_and_improvements** | Value of buildings and improvements. |
| **machinery_furniture_equipment** | Value of machinery, furniture, and equipment. |
| **other_properties** | Other real estate properties. |
| **leases** | Right-of-use assets from leases. |
| **accumulated_depreciation** | Total depreciation recorded against PPE assets to date. |
| **goodwill_and_other_intangible_assets** | Sum of Goodwill and other non-physical assets. |
| **goodwill** | Premium paid for an acquisition over the fair value of net assets. |
| **other_intangible_assets** | Non-physical assets like patents, trademarks, and copyrights. |
| **investments_and_advances** | Long-term investments and advances to others. |
| **long_term_equity_investment** | Equity stakes in other companies held for the long term. |
| **investmentin_financial_assets** | Investments in financial instruments. |
| **financial_assets_designatedas_fair_value_through_profitor_loss_total** | Financial assets recorded at fair value with changes flowing to profit/loss. |
| **available_for_sale_securities** | Debt or equity securities not classified as held-for-trading or held-to-maturity. |
| **financial_assets** | General category for financial assets. |
| **non_current_deferred_assets** | Long-term deferred charges or assets. |
| **non_current_deferred_taxes_assets** | Future tax benefits to be realized beyond one year. |
| **other_non_current_assets** | Miscellaneous long-term assets. |
| **total_liabilities_net_minority_interest** | Total liabilities excluding minority interest. |
| **current_liabilities** | Obligations due within one year. |
| **payables_and_accrued_expenses** | Money owed to suppliers and accrued liabilities. |
| **payables** | General category for money owed. |
| **accounts_payable** | Money owed to suppliers for goods/services received on credit. |
| **total_tax_payable** | Total taxes currently owed to authorities. |
| **income_tax_payable** | Income tax obligations currently due. |
| **other_payable** | Miscellaneous payables. |
| **current_accrued_expenses** | Expenses incurred but not yet paid (current). |
| **pensionand_other_post_retirement_benefit_plans_current** | Current portion of pension and post-retirement obligations. |
| **current_debt_and_capital_lease_obligation** | Debt and lease obligations due within one year. |
| **current_debt** | Short-term borrowings and current portion of long-term debt. |
| **commercial_paper** | Unsecured, short-term debt instrument issued by the corporation. |
| **other_current_borrowings** | Miscellaneous short-term borrowings. |
| **current_deferred_liabilities** | Obligations for revenue received but not yet earned (deferred revenue). |
| **current_deferred_taxes_liabilities** | Taxes owed but postponed for payment within the year. |
| **current_deferred_revenue** | Short-term unearned revenue. |
| **other_current_liabilities** | Miscellaneous short-term liabilities. |
| **total_non_current_liabilities_net_minority_interest** | Long-term liabilities excluding minority interest. |
| **long_term_debt_and_capital_lease_obligation** | Debt and lease obligations due after one year. |
| **long_term_debt** | Loans and financial obligations lasting over one year. |
| **long_term_capital_lease_obligation** | Long-term obligations under capital leases. |
| **non_current_deferred_liabilities** | Long-term deferred revenue or other liabilities. |
| **non_current_deferred_taxes_liabilities** | Taxes owed but postponed for payment beyond one year. |
| **non_current_deferred_revenue** | Long-term unearned revenue. |
| **tradeand_other_payables_non_current** | Long-term trade payables. |
| **other_non_current_liabilities** | Miscellaneous long-term liabilities. |
| **total_equity_gross_minority_interest** | Total equity including minority interest. |
| **stockholders_equity** | Equity attributable to the company's shareholders. |
| **capital_stock** | Total par value of shares issued. |
| **preferred_stock** | Equity ownership with preferential claims on dividends and assets. |
| **common_stock** | Equity representing ownership in the company. |
| **additional_paid_in_capital** | Capital paid by investors in excess of par value. |
| **retained_earnings** | Cumulative net income retained in the business (not distributed as dividends). |

| **common_stock_equity** | Value of common stock in equity. |
| **capital_lease_obligations** | Total obligations under capital leases. |
| **net_tangible_assets** | Total assets minus intangible assets and liabilities. |
| **working_capital** | Current Assets minus Current Liabilities. |
